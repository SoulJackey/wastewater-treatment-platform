# 隧道废水处理智能平台 - HTML原型Demo

## 项目简介

这是一个为工程建设项目生产生活废污水智能平台创建的HTML原型Demo，涵盖生活污水、隧道废水、拌合站废水、砂石废水四类处理系统的管控功能。

## 技术栈

- **前端框架**: 纯HTML + CSS + JavaScript（无框架）
- **图表库**: ECharts 5.4.3
- **地图组件**: Leaflet.js + OpenStreetMap（无需API密钥）
- **图标库**: Font Awesome 6.4.0
- **数据存储**: 模拟数据（JS对象）

## 功能特性

### 1. 综合看板
- **公司级看板**: GIS地图展示所有项目分布、项目列表卡片、设备类型统计、关键指标、报警统计、能耗趋势
- **项目级看板**: 项目基本信息、设备运转时长趋势、能耗趋势、设备状态概览、实时报警列表

### 2. 设备管理
- **设备档案**: 设备列表展示、搜索筛选、设备详情查看、设备状态管理

### 3. 在线监控
- **实时监控**: 设备实时数据展示、实时曲线图、设备启停控制
- **历史数据**: 历史数据查询、曲线图展示、数据导出
- **报警管理**: 报警列表、报警筛选、报警处理
- **视频监控**: 视频播放器、云台控制
- **数字孪生**: 设备平面布局图、实时状态显示

### 4. 统计分析
- **数据报表**: 日报/周报/月报/年报、报表预览和导出
- **能耗统计**: 能耗总览、能耗趋势、设备能耗排名

### 5. 系统管理
- **人员管理**: 用户列表、角色权限、用户管理

## 开发说明

### 添加新页面

1. 在 `pages/` 目录创建新的HTML文件
2. 复制侧边栏和顶部栏代码
3. 根据需要调整菜单高亮
4. 引入必要的CSS和JS文件

### 修改样式

- 通用样式: 修改 `css/common.css`
- 布局样式: 修改 `css/layout.css`
- 模块样式: 修改 `css/modules.css`

### 修改数据

编辑 `js/data.js` 文件中的数据对象。

## 注意事项

1. **地图功能**: 使用OpenStreetMap，无需API密钥，但需要网络连接
2. **图表渲染**: ECharts需要联网加载CDN资源
3. **图标显示**: Font Awesome需要联网加载CDN资源
4. **浏览器缓存**: 修改代码后建议清除浏览器缓存或使用Ctrl+F5刷新

## 常见问题

### Q: 地图不显示？
A: 检查网络连接，确保可以访问OpenStreetMap服务。

### Q: 图表不显示？
A: 检查ECharts CDN是否加载成功，打开浏览器控制台查看是否有错误。

### Q: 如何修改默认数据？
A: 编辑 `js/data.js` 文件，修改相应的数据对象。

### Q: 如何集成真实后端API？
A: 在 `js/common.js` 中的 `ajaxRequest` 函数中添加真实的API调用逻辑。
