# 快速启动指南

## 方法1: 直接打开（最简单）

1. 双击 `index.html` 文件
2. 文件会在默认浏览器中打开
3. 开始使用

**注意**: 直接打开可能存在一些跨域限制，推荐使用方法2。

## 方法2: 本地服务器（推荐）

### 使用Python（推荐）
```bash
# 在项目目录下执行
cd "D:\1. 工作\1. 水利项目\B2. 隧道废水处理系统\CC"
python -m http.server 8000
```
然后访问: http://localhost:8000

### 使用Node.js
```bash
# 全局安装http-server
npm install -g http-server

# 启动服务器
http-server -p 8000
```
然后访问: http://localhost:8000

### 使用PHP
```bash
php -S localhost:8000
```
然后访问: http://localhost:8000

## 主要功能演示

### 1. 查看GIS地图
1. 进入"公司级看板"
2. 查看地图上的项目标记
3. 点击标记查看项目详情
4. 点击"查看详情"进入项目看板

### 2. 查看设备列表
1. 进入"设备档案"
2. 使用搜索框搜索设备
3. 使用筛选器过滤设备
4. 点击"查看"按钮查看设备详情

### 3. 实时监控
1. 进入"实时数据"
2. 在左侧选择要监控的设备
3. 查看实时数据面板（每3秒自动刷新）
4. 查看实时曲线图
5. 使用控制面板开关设备

### 4. 处理报警
1. 进入"报警管理"
2. 查看报警统计卡片
3. 使用筛选器查找特定报警
4. 点击"查看"查看报警详情
5. 点击"处理"标记报警为处理中

### 5. 导出数据
1. 在任何有数据的页面
2. 点击"导出"按钮
3. 数据会导出为CSV文件

## 常见问题

### Q: 地图不显示？
A: 检查网络连接，确保可以访问OpenStreetMap。

### Q: 图表不显示？
A: 打开浏览器控制台（F12）查看错误，确保CDN资源加载成功。

### Q: 数据不刷新？
A: 检查浏览器控制台，可能是JavaScript错误。

### Q: 样式错乱？
A: 清除浏览器缓存（Ctrl+F5）。
